# 个人社交名片生成器

一个现代化的个人社交名片生成器，帮助用户快速创建专业、美观的个人名片。

## 功能特点

### 🎨 现代简约设计
- 采用现代简约风格，信息层次清晰
- 视觉重点突出，干净简洁
- 响应式设计，适配各种设备

### 📋 完整的信息结构

#### 头部信息
- 头像展示区域（支持文件上传和URL链接）
- 姓名显示
- 地点信息
- 身份标签

#### 主体部分
- **履历亮点** - 展示个人主要成就
- **擅长领域** - 显示专业技能
- **兴趣爱好** - 个人兴趣展示
- 每个部分都配有相应图标，增强可读性

#### 页脚部分
- 个人态度/座右铭展示
- 二维码展示区域（支持上传和生成）

### ⚡ 交互功能
- **实时预览** - 输入信息时实时更新名片预览
- **一键生成** - 点击按钮即可生成完整名片
- **示例数据** - 提供示例数据快速体验
- **文件上传** - 支持头像和二维码图片上传
- **PNG下载** - 高清PNG格式名片下载
- **二维码生成** - 自动生成链接二维码
- **名片分享** - 生成名片二维码，扫描即可查看
- **链接分享** - 一键复制名片分享链接
- **键盘快捷键** - 支持 Ctrl+Enter 生成，Ctrl+S 下载，Ctrl+Q 生成二维码

## 技术栈

- **HTML5** - 页面结构
- **Tailwind CSS** - 样式框架
- **JavaScript** - 交互逻辑
- **Font Awesome** - 图标库
- **html2canvas** - PNG图片生成
- **QRCode.js** - 二维码生成

## 使用方法

1. 打开 `index.html` 文件
2. 在左侧表单中填写个人信息：
   - 基本信息（姓名、地点、身份标签）
   - 头像（支持文件上传或URL链接）
   - 履历亮点
   - 擅长领域
   - 兴趣爱好
   - 个人态度
   - 二维码（支持文件上传或链接生成）
3. 点击"生成名片"按钮或使用快捷键 `Ctrl+Enter`
4. 在右侧查看生成的名片预览
5. 使用功能按钮：
   - "下载PNG名片" - 保存高清名片图片
   - "生成名片二维码" - 创建可扫描查看的名片二维码
   - "分享名片链接" - 复制名片分享链接到剪贴板

## 快捷键

- `Ctrl + Enter` - 生成名片
- `Ctrl + S` - 下载PNG名片
- `Ctrl + Q` - 生成名片二维码

## 特色功能

### 🔄 实时预览
用户在输入信息时，右侧的名片预览会实时更新，提供即时的视觉反馈。

### 📱 响应式设计
采用 Tailwind CSS 的响应式网格系统，在不同设备上都能完美显示。

### 🎯 用户体验优化
- 表单验证和提示
- 平滑滚动动画
- 悬停效果和过渡动画
- 直观的图标指引
- 实时通知反馈

### 🖼️ 头像支持
- 支持本地文件上传（最大5MB）
- 支持通过 URL 链接添加
- 自动适配显示尺寸

### 📱 二维码功能
- 支持上传自定义二维码图片
- 支持根据链接自动生成二维码
- 一键生成名片分享二维码
- 扫描二维码即可查看完整名片

### 💾 高级下载
- 使用html2canvas生成高清PNG图片
- 支持下载名片二维码
- 文件自动命名（姓名-个人名片.png）

### 🔗 分享功能
- 生成包含名片数据的分享链接
- 一键复制到剪贴板
- 通过URL参数自动加载分享的名片

## 浏览器兼容性

支持所有现代浏览器：
- Chrome 60+
- Firefox 60+
- Safari 12+
- Edge 79+

## 开发说明

本项目使用纯前端技术开发，无需服务器环境，可直接在浏览器中运行。所有样式基于 Tailwind CSS CDN，图标使用 Font Awesome CDN。
